# kt5

Это демонстрационный проект для изучения семантического версионирования (SemVer) и управления релизами.

# Система версионирования

Проект использует [Semantic Versioning 2.0.0](https://semver.org/):

- MAJOR версия (X.0.0): обратно несовместимые изменения
- MINOR версия (0.X.0): новая функциональность с обратной совместимостью
- PATCH версия (0.0.X): исправления багов с обратной совместимостью

# Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/HamisLord/kt5.git

# Перейти в папку проекта
cd kt5

# Запустить приложение
java main.java
